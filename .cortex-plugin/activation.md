This plugin provides Clinical Document Intelligence for Snowflake — AI-powered extraction, search, and governance for clinical documents (discharge summaries, radiology reports, pathology reports).

Use when: clinical documents, PDF extraction, document classification, AI_PARSE_DOCUMENT, clinical search, HIPAA governance, medical transcription, discharge summary, radiology report, pathology report.

Run `/clinical-docs:deploy` to set up the pipeline in your Snowflake account.
